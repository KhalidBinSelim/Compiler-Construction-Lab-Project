# ByteBuilders Mini Language Compiler

A mini programming language compiler front-end developed using Flex, Bison, and C/C++.  
This project is being developed for the Compiler Construction Laboratory course at Metropolitan University, Bangladesh.

## Project Objective

The compiler will process source programs through the following phases:

1. Lexical Analysis
2. Syntax Analysis
3. Abstract Syntax Tree Construction
4. Symbol Table Management
5. Semantic Analysis
6. Three Address Code Generation
